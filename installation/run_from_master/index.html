Run from master

If you want to build a package yourself, or contribute - here is a guide for how to do that.

Dependencies

  • NodeJS LTS

Building plugin

npm install -g yarn
yarn install --pure-lockfile
yarn build

To build plugin and rebuild on file change

yarn watch

Run tests

yarn test

Run tests on file change

yarn jest
